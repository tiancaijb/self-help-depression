# 注意力涣散 · 工具与方法

## 抑郁性注意力涣散（≠ ADHD）

抑郁导致的注意力问题不是"注意力的硬件坏了"，是**没电了**。大脑被反刍和负面情绪占用了大量带宽，剩余给当前任务的认知资源不够了。

| ADHD | 抑郁性注意力涣散 |
|------|----------------|
| 从小就有 | 随抑郁发作而出现 |
| 注意力不稳定（雷达式扫射） | 注意力转不动（燃油不足） |
| 有能量但控制不了方向 | 没有启动的能量 |
| 刺激寻求 | 刺激回避 |

如果怀疑自己有 ADHD → 见 `05-adhd-improve-focus.txt` 文字稿。

## 最小行动法（Feeling Good Ch5）

核心原则：**把任务缩到几乎不需要注意力也能做的大小。**

### 反拖延表（核心工具）

对拖着不做的事：

| 步骤 | 预测难度(0-100) | 预测满足感(0-100) | 实际难度 | 实际满足感 |
|------|---------------|-----------------|---------|---------|
| 拆成≤15 分钟的步骤 | | | | |

规律：**实际难度几乎总是远低于预测，实际满足感几乎总是远高于预测。**

### 完美主义（Feeling Good Ch14）

完美主义的本质："我必须做到完美才有价值。"

**反完美主义表：**
列出"如果做差，实际会怎样"——通常最坏情况概率 < 10%，而你现在付出的代价（拖延/焦虑）每天都在发生。

**开心预测法：**
做一件事前预测享受程度（0-10），做完后记实际享受程度。通常实际 > 预测。这证明你的大脑系统性地低估了做事带来的满足感。

**灰色思维：**
用百分比替代黑白判断。"这次演讲 70% OK，30% 要改进"而不是"彻底失败"。

### 碎步法
任何任务拆成 ≤15 分钟能做完的块。15 分钟到了不想继续就停——"做了比没做好"。

### 每日活动计划表
每小时写计划 → 做完后打分 M（掌控感）和 P（愉悦感），0-5 分。
- 如果什么都不做，你会被负面想法淹没
- 做一点事（任何事）就能打断这个循环
- 重点：**做了就算数，不看做得多好**

### 拖延的 13 种心理定势
Feeling Good 列出 13 种拖延背后的思维模式，最常见：
- **低挫折耐受力**："我应该轻松搞定"→ 换"难是正常的"
- **恐惧失败**："不做就不会输"→ 换"做了就是赢"
- **强制与怨恨**："我必须做"→ 换"我选择做"
- **低估回报**："做了也没意思"→ 先做再打分，用数据反驳

## 多巴胺与动机（Huberman 06）

- 多巴胺不是愉悦分子，是**渴望/动机分子**
- 注意力涣散时，不是"不想做"，是**大脑没有足够的 dopamine 驱动行动**
- 多巴胺基线的重置方法 → 见 `dopamine.md`

### 声音工具（Huberman 03）
- 白噪音/棕色噪音 → 对注意力涣散有帮助
- 双耳节拍（binaural beats）→ 40Hz 用于专注
- 咖啡 + 白噪音的组合效果更好

### 咖啡因管理
- 醒来后 90-120 分钟才喝咖啡（避免午后崩溃）
- 下午 2 点后不喝
- 咖啡因+光照的协同效果

### 分心管理
- 专注 vs 分心是两套独立的神经回路（Huberman 05）
- 减少分心 ≠ 提高专注，需要分别训练
- 手机放另一个房间、关闭通知、用物理屏障

## 原子习惯（James Clear）

抑郁期最实用的 3 条：

1. **2 分钟规则**：任何习惯缩减到 2 分钟。"每天跑步"→"穿上跑鞋"
2. **习惯堆叠**：绑定到已有习惯。"喝完咖啡后→写 1 分钟思维记录"
3. **环境设计**：让想做的事极其容易（跑鞋放床边），想戒的事极其困难（手机放远）

## 注意力的底线

整个 skill 的核心理念——**注意力不是靠"练"回来的，是靠"做的事足够小"，小到不需要注意力也能做完。** 如果一件事需要意志力才能开始，它太大了。
