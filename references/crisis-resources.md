# 危机资源 · 备而不用

**这个文件平时不用读。只有当出现自杀念头、自伤念头、严重自毁行为时才打开。**

## 紧急情况判断

如果出现以下任何一条，立刻寻求专业帮助：

- 有自杀的想法 + **有具体计划**（怎么死、什么时候、在哪里）
- 有自杀的想法 + **有实施手段**（已经拿到药/工具）
- 有伤害他人的想法或计划
- 出现幻觉（听到别人听不到的声音、看到别人看不到的东西）
- 严重自残行为

## 立即行动清单（按顺序）

### 1. 安全第一
- **去除身边的危险物品**（药物/刀具/绳索等）
- 不要独处——去有人/有人的地方（客厅/便利店/街上）
- 如果一个人在房间里，打开门，去公共空间

### 3. 联系一个信任的人
- 家人/朋友/同事
- 不要说"没事"——直接说"我现在需要帮助"
- 让这个人来陪你，或你过去找他

### 4. 去最近的医院急诊
- 任何综合医院都有急诊科
- 告诉医生"我需要心理方面的帮助"
- 不要自己开车——打车或让人送

## 如果你在帮助别人

如果对方（朋友/读者/来访者）表现出自杀意图：

1. **不承诺保密**：直接说"我很担心你的安全，这件事我不能保密"
2. **直接问**："你有自杀的想法吗？有计划吗？有手段吗？"——问自杀念头不会"植入"念头，反而降低风险
3. **不留独处**：在专业帮助到达前不让对方独处
4. **移除危险物品**
5. **陪对方拨打热线或去急诊**

## 如果自己处于危机但还没到上面那条线

你不需要"够格"才能求帮助。只要你在想"我是不是应该打那个电话"——就打。
